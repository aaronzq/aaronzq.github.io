Build Python-Opencv Environment

22 Oct, 2017 · Read in about 2 min · (220 Words)

On your Raspberry Pi

  • Check your cpu information: more /proc/cpuinfo
    We may get the model name of the processor: ARMv7 Processor rev 4 (v7l)
  • Check which version of Python your pip targets: pip -V
    In default, pip will connect to python2.7 while pip3 connects to python3. If your pip is bound to python3 and you want to redirect it to python3. Try: which pip Locate the pip at /home/aaronrasp/.local/bin/pip and nano /home/aaronrasp/.local/bin/pip Change the heading #!/usrbin/python3 to #!/usrbin/python
    Check again pip -V
    Done.
  • Upgrade your pip pip install -U pip

For an easy install

Simply grab the python-opencv package. In your console, type:

$ sudo apt-get install python-opencv

A non-latest-version pre-compiled opencv will be installed under Python2.7.

For a better install

$ sudo apt-get install build-essential git cmake pkg-config

$ sudo apt-get install libjpeg-dev libtiff5-dev libjasper-dev libpng12-dev

$ sudo apt-get install libavcodec-dev libavformat-dev libswscale-dev libv4l-dev
$ sudo apt-get install libxvidcore-dev libx264-dev

$ sudo apt-get install libgtk2.0-dev

$ sudo apt-get install libatlas-base-dev gfortran

$ sudo apt-get install python2.7-dev python3-dev

$ cd ~
$ wget -O opencv.zip https://github.com/opencv/opencv/archive/3.3.1.zip
$ unzip opencv.zip

$ pip install numpy

$ cd ~/opencv-3.0.0/
$ mkdir build
$ cd build
$ cmake -D CMAKE_BUILD_TYPE=RELEASE \
	-D CMAKE_INSTALL_PREFIX=/usr/local \
	-D INSTALL_C_EXAMPLES=OFF \
	-D INSTALL_PYTHON_EXAMPLES=ON \
	-D OPENCV_EXTRA_MODULES_PATH=~/opencv_contrib-3.0.0/modules \
	-D BUILD_EXAMPLES=ON ..

$ make -j4

$ make clean
$ make
